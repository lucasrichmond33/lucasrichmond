1.1.9 - Minion Asteroids



Minion Asteroids is a game using several conditions to achieve a desired output.

1.2.5 - Rock, Paper, Scissors!



"Rock, Paper, Scissors!" is the same game you have been playing since you were a kid but in the format of an old Pokemon game!
In the program you type the input Rock(1), Paper(2), or Scissors(3) and the code will generate an opposing output.
Depending on the result of the game, a win or lose animation will generate on the screen.

3.1.6 Project - Bring Back The Pandas!



"Bring Back The Pandas!" is a breeeding simulator where the user can repopulate the bamboo forests!
In the program the user inputs a name for the daddy panda where a name will be presented for a few seconds and from there just start clicking!
Click the adult panda to spawn a baby panda and click the baby pandas to grow them to full adults and just keep clicking until the panda counter is at your desired number!
